= OpenApi 3 Plugin

openapi-jaxb plugin

== Maven central

This plugin is available in maven central as:
[source,xml]

----
    <dependency>
        <groupId>hu.icellmobilsoft.jaxb</groupId>
        <artifactId>openapi-jaxb-plugin</artifactId>
        <version>${version}</version>
    </dependency>
----
